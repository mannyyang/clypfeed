Claude Desktop v1.1.5749 ships native computer use with full desktop control APIs

The latest official Claude Desktop build quietly added computer use support via a new @ant/computer-use-mcp dependency, enabling programmatic control of screenshots, mouse, and keyboard input. A new ComputerUseTcc IPC API also landed, exposing six macOS TCC permission management methods for accessibility and screen recording — the plumbing needed for Claude to interact directly with the user's desktop. The addition follows Anthropic's acquisition of Vercept in late February, which was specifically aimed at advancing Claude's computer use capabilities.
