Anthropic Commits $100M to Claude Partner Network

Anthropic announced a $100 million investment into the newly launched Claude Partner Network, a formal program for organizations helping enterprises adopt Claude. The network includes a new Claude Certified Architect credential, co-marketing funds, dedicated Applied AI engineers for live deals, and a Code Modernization starter kit — with Accenture already training 30,000 employees. Anthropic is also scaling its partner-facing team fivefold, signaling an aggressive push to dominate enterprise AI deployment channels in 2026.

Claude Now Generates Interactive Charts and Visualizations Inline

Anthropic launched an interactive visuals feature in beta across all Claude plans, including free. Claude can now build custom charts, diagrams, and data visualizations that appear inline within conversations and update dynamically as the discussion evolves — distinct from the existing side-panel artifacts system. The feature is on by default and was previewed last fall as 'Imagine with Claude.'

Anthropic Productvia GitHub

Claude Code v2.1.74 Ships Context Optimizer and Critical Memory Leak Fix

Claude Code v2.1.74 adds actionable suggestions to the /context command — automatically identifying context-heavy tools, memory bloat, and capacity warnings with optimization tips. A critical fix addresses unbounded RSS memory growth caused by unreleased streaming API response buffers on the Node.js/npm code path. The release also resolves MCP OAuth hangs, RTL text rendering failures on Windows, LSP breakage on Windows, and voice mode microphone permission issues on macOS.

Musk Unveils Macrohard — Joint Tesla-xAI Project to Automate Software Companies

Elon Musk revealed 'Macrohard' (also called 'Digital Optimus'), a joint Tesla-xAI project that pairs Grok with a Tesla-built AI agent capable of processing real-time screen video and controlling keyboard and mouse to emulate software company workflows. Anthropic Launches the Anthropic Institute, Led by Co-Founder Jack Clark

Anthropic announced The Anthropic Institute, a new internal research body focused on studying the societal impacts of powerful AI. Led by co-founder Jack Clark (now Head of Public Benefit), it merges three existing teams — the Frontier Red Team, Societal Impacts, and Economic Research — and is hiring economists, ML engineers, and social scientists. The Institute reflects Anthropic's belief that transformative AI is imminent and that independent, candid research into its risks is urgently needed.

Competitivevia r/ClaudeAI

Microsoft Launches Copilot Cowork — an Agentic M365 Assistant Powered by Anthropic's Claude

Microsoft unveiled Copilot Cowork, a new agentic AI layer embedded in Microsoft 365 that autonomously executes multi-step tasks across Outlook, Teams, Excel, and PowerPoint — then checks in before applying changes. The product is built on Anthropic's Claude, extending the same agentic architecture behind Claude Cowork (launched January 2026) into Microsoft's enterprise data graph. Pricing starts at $30/month in the M365 Copilot plan, with full access in a new $99/month E7 Frontier Suite launching May 1.

Anthropic Productvia Claude Code GitHub Releases

Claude Code v2.1.73 Ships with /btw Overlay, Model Overrides, and 15+ Bug Fixes

Anthropic released Claude Code v2.1.73, a substantial update that includes a new /btw command (introduced in v2.1.72) that lets users ask quick side questions without interrupting a running session or polluting the context window. The update also adds a modelOverrides setting for custom provider model IDs (e.g., Bedrock inference profile ARNs), defaults Bedrock/Vertex/Foundry to Opus 4.6, and fixes over 15 bugs including CPU-loop freezes, subagent cleanup failures, and Linux sandbox issues.

Yann LeCun's AMI Labs Raises $1B Seed at $3.5B Valuation to Build World Models

Meta's Chief AI Scientist Yann LeCun formally unveiled Advanced Machine Intelligence (AMI Labs), a new AI startup that raised a $1.03B seed round — reportedly one of the largest ever — at a ~$3.5B pre-money valuation. The company, co-founded with CEO Alex Lebrun and CSO Saining Xie, aims to build AI that understands the physical world through JEPA-based world models, challenging the dominant language-scaling paradigm. French President Macron and European media hailed it as a major milestone for European AI.

Anthropic announced it will open a new office in Sydney, Australia in the coming weeks — its fourth Asia-Pacific location alongside Tokyo, Bengaluru, and Seoul.
